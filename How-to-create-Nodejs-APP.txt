Step1. Create a new directory
Step2. cd into the new directory
Step3. Create a new file called app.js
Step4. Inialize npm with app.js as start point   
       command:  npm init
Step5. Install express
	command: npm install express
Step6. Install body-parser
	command: npm install body-parser
Step7. Run server: 
	node app.js
	or nodemon app.js
       
        Install nodemon:
	npm install -g nodemon
